# **Connect four game with AI**
**Florian Hauber**
---

This is a classic game of connect four where you play against some sort of AI. The basic game logic was designed by me but gemini helped me with some of the AI components in this code. 

**for the MAX_DEPTH i would recommend something from 5-7 for good gameplay but not thinking too long**

## Downloading
---
If you want to download this repo just download the .zip file unpack it and run it (Dotnet version 10.0). And you should be good to go
